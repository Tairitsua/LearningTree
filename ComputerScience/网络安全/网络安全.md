# 网络安全

本目录将 [网络安全（旧）](../网络安全（旧）.md) 与 [网络安全（新）](../网络安全（新）.md) 合并重组为一套更适合长期维护的笔记。整理时尽量保留原始知识点，只修复标题层级、列表格式、代码块、链接格式，以及少量明显的事实或拼写错误。

## 学习路径

1. 先读 [基础知识](基础知识.md)，统一 `HTTP`、`URL`、`DOM`、`BOM`、编码与常见术语。
2. 再读 [常见Web漏洞](常见Web漏洞.md)，建立 `XSS`、`CSRF`、点击劫持、开放重定向的整体认识。
3. 然后读 [文件上传与命令执行](文件上传与命令执行.md) 与 [SQL注入与数据库](SQL注入与数据库.md)，进入常见利用与绕过思路。
4. 最后补 [Web服务器与表单基础](Web服务器与表单基础.md) 和 [PHP基础](PHP基础.md)，把攻防中经常遇到的开发侧知识补齐。

## 整理原则

1. 文件标题统一作为唯一 `H1`。
2. 章节按“基础概念 -> 漏洞类型 -> 利用与绕过 -> 支撑知识”组织。
3. 专有英文名词、命令、代码、协议字段统一使用反引号或代码块包裹。
4. 原文中的重复内容尽量只保留一份，减少后续维护成本。
